Converting, Downloading, & Hosting

  • Sometimes you want to use a particular font, but it's not available on Google Fonts, or you can't find a web type version.
  • The solution is to use FontSquirrel to convert your fonts and host them yourself.

Currently this element has no special font attached, but let's add one!

  • Find your font on FontSquirrel or wherever it is you find fonts (we will talk about DaFont later..........)
  • Go to the Generator page
  • Run the generator & download the results
  • Copy the fonts into your fonts directory & the CSS into your CSS file.
  • Important! Make sure you're accurately pointing your CSS rule to the correct directory!!!
